# iView Chrome Extension

本脚手架集成目前比较流行的 iView，一套高质量的 UI 组件库。

#配置信息
软件运行配置说明请看 [config.js](config.js)

**安装依赖：**

```bat
npm install
```

**以 dev 方式运行：**

```bat
npm run dev
```

**以本地方式运行：**

```bat
npm run local:dev
```

**打包成可执行文件：**

```bat
npm run build
```
